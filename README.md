**Neuro-TOiR** — это высокотехнологичная цифровая платформа для автоматизации процессов технического обслуживания, предиктивной диагностики и ремонта (ТОиР) железнодорожного транспорта и инфраструктуры промышленных предприятий.

Система объединяет в себе возможности телематического контроля, искусственного интеллекта и современных веб-технологий для сквозного мониторинга и снижения издержек.

---

##  Ключевые модули и интеллектуальные функции

###  Модуль искусственного интеллекта (AI )
* **Предиктивная аналитика:** Алгоритмы машинного обучения прогнозируют износ элементов инфраструктуры и узлов подвижного состава, позволяя проводить ремонт до возникновения аварийных ситуаций.
* **Анализ аномалий:** Интеллектуальный контроль расхода ресурсов (топлива, смазочных материалов) и автоматическое выявление сливов или перерасхода.

###  Телематическая интеграция с Wialon API
* **Real-time GPS/ГЛОНАСС мониторинг:** Сквозная интеграция с платформой Wialon позволяет в реальном времени отслеживать точное местоположение, скорость и направление движения тепловозов и спецтехники на путях предприятия.
* **Сбор датчиков:** Автоматический импорт данных из Wialon (уровень топлива в баке, температура двигателя, моточасы, нагрузка на оси) непосредственно в аналитическую базу данных системы.
* **Геозоны и маршруты:** Контроль соблюдения скоростного режима на специализированных участках путей и фиксация простоев техники.

## Стек технологий

* **Frontend:** Next.js (App Router), React, TypeScript, Tailwind CSS
* **Компоненты UI:** Radix UI, Lucide Icons, Recharts (интерактивные аналитические дашборды)
* **Backend & Database:** PostgreSQL / MS SQL Server, Supabase (управление связями и real-time подписки)
* **AI & Data Science Модуль:** Python (среда Anaconda), библиотеки машинного обучения (обработка данных датчиков и детекция дефектов)
* **Интеграции:** Wialon REST API / SDK для обработки телематических данных

## Перспективы развития
* Создание полноценного "цифрового двойника" (Digital Twin) железнодорожной сети предприятия.
* Повышение точности нейросетевых моделей за счет расширения базы данных паттернов износа.
